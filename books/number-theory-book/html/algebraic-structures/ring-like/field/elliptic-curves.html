Elliptic curves

The points of an elliptic curve form a group under point addition.

Point addition

Point addition in elliptic curves isn't the usual arithmetic addition that we know. See Andrea Corbellini's Interactive Tool for Elliptic Curve.

Elliptic curve cryptography

In elliptic curve cryptography, the difficulty of ECDLP relies on the properties of the group under point addition.

One example is to count the points on the elliptic curve.

Elliptic curve forms

  • Weierstrass form

    y2=x3+ax+b

  • Montgomery form

    by2=x3+ax2+x

  • Edwards form

    x2+y2=1+dx2y2

  • Twisted Edwards form

    ax2+y2=1+dx2y2

Example 1

Simple example of an elliptic curve over a finite field F7​, i.e. the field of integers modulo 7.

This means the field consists of elements {0,1,2,3,4,5,6}.

Define an elliptic curve over F7​:

y2≡7​x3+2x+1

The set of points (x,y) that satisfies this equation (along with a special point of infinity) forms the elliptic curve over F7​.

To get this set of points, we need to cycle through values of x∈F7​ to get the corresponding y.

For example, when x=1, we have:

y2y2​≡7​13+2×1+1≡7​4​​

What are the values of y∈F7​ that satisfy this congruence? Let's enumerate them:

  • When y=0, 02≡0≡7​4
  • When y=1, 12≡1≡7​4
  • When y=2, 22≡4≡7​4
  • When y=3, 32≡9≡7​4
  • When y=4, 42≡16≡7​4
  • When y=5, 52≡25≡7​4
  • When y=6, 62≡36≡7​4

Therefore, when x=1, y=2,5.
