# Preface

## Notations

### Prime numbers

Prefer using $p$ to represent a prime number.

### Modulus

Prefer using $n$ to represent a modulus.

### Positive integer

Prefer using $a$ for positive integers.

### Congruence

The equivalence relation

$$
a \equiv b (\bmod n)
$$

is written as

$$
a \equiv_n b
$$
