# PKCS #1

[RFC 8017](https://datatracker.ietf.org/doc/html/rfc8017)

PKCS #1 is used for RSA only. Note that PKCS #8 can also apply to RSA.

PEM-encoded PKCS #1 RSA private key:

```
-----BEGIN RSA PRIVATE KEY-----
MIIEpAI....
....
-----END RSA PRIVATE KEY-----
```
