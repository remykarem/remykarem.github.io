# X.509

[International Telecommunication Union X.509 standards](https://www.itu.int/rec/T-REC-X.509/en), [RFC 5280](https://datatracker.ietf.org/doc/html/rfc5280)

Standard format for public key certificates

X.509
- public key
- Signature
- Other info
    - Country name
    - Website
