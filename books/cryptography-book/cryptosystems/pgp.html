PGP

Pretty Good Privacy

RFC 4880

PGP is a hybrid encryption system for encrypting data. It uses both a symmetric and asymmetric key.

PGP is often associated with sending encrypted emails and encrypting files.

PGP also uses a decentralised trust model called Web of Trust.

Encryption:

(message, key)→ciphertext

(key, public key)→encrypted key

Decryption:

(encrypted key, private key)→key

(key, ciphertext)→message

Warning

PGP is an insecure and outdated ecosystem that hasn’t reflected cryptographic best practices in decades.

Source: https://blog.yossarian.net/
