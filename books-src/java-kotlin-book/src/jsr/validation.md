# Validation

## Bean Validation

[JSR-303](https://jcp.org/en/jsr/detail?id=303) (Bean Validation) and [JSR-380](https://jcp.org/en/jsr/detail?id=380) (Bean Validation 2.0)

package `javax.validation`

* General — `@NotEmpty`, `@NotBlank`
* Numbers — `@Min`
* Dates — `@Future`, `@Past` etc.
* Emails — `@Email`
* Collections — `@Size`
