# Tree

- Extension of linked list

Types of nodes:
* root
* internal
* leaf

Characteristics:
* No. of children: **_m_-ary** tree is one where each node must have a maximum of _m_ children
* **Balanced**
* **Leaf-similar**

- “level”, “parent”-“child”, “height”, “depth”
- All nodes must be connected. There must not be any cycles.
