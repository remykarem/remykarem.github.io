# Commutative ring

A **commutative ring** satisfies the following axioms:

|                | Addition | Multiplication |
|----------------|----------|----------------|
| Closure        | ✅       | ✅             |
| Associativity  | ✅       | ✅             |
| Identity       | ✅       | ✅             |
| Invertibility  | ✅       |                |
| Commutativity  | ✅       | ✅             |
| Distributivity | -        | ✅             |
