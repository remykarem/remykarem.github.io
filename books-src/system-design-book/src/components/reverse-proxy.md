# Reverse proxy (load balancer)

A **reverse proxy** is commonly used for:
* [**Load balancing**](./load-balancing.md)
* [**Identity hiding**](./identity-hiding.md) of the backend servers
* [Routing](./routing.md)
* [Rate limiting](./rate-limiting.md)
* [Caching](./caching.md)
* [SSL termination](./ssl-encryption-and-decryption.md) for backend servers
* Serve static content via [caching](./caching.md)
