# Webhook vs. polling

|                        | Webhook         | Polling / REST API |   |   |
|------------------------|-----------------|--------------------|---|---|
| Resource               | Fewer resources | A lot of resources |   |   |
| Near real-time updates | Yes             | No                 |   |   |
