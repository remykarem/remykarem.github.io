# Rate limiting

aka throttling

Control the rate at which requests are made to prevent abuse or overload.

Algorithms
* Fixed window
* Sliding window
* Token bucket

Helps with:
* [Performance](../goals/performance.md)
* [Availability](../goals/availability.md)
* [Reliability](../goals/reliability.md)
