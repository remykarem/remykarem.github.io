# ASCII

aka US ASCII

American Standard Code for Information Interchange

This is a **7-bit** character set.

Character set:

* English characters

  | Symbol | Hex |
  |--------|-----|
  | `0`    | 30  |
  | `1`    | 31  |
  | ..     | ..  |
  | `A`    | 41  |
  | `B`    | 42  |
  | ..     | ..  |
  | `a`    | 61  |
  | `b`    | 62  |

* Control characters

~~~admonish note title="Storage"
The character set can be encoded using a storage size of 1 byte.
~~~
