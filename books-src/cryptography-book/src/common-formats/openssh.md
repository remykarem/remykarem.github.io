# OpenSSH

[RFC 4253](https://datatracker.ietf.org/doc/html/rfc4253)

The OpenSSH format is used to represent private and public keys that are used for SSH authentication.

(Not sure how they encode the keys.)

```txt
ssh-rsa AAAAB...
```
